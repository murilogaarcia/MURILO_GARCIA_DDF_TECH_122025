# Arquitetura da Solução

## Visão Geral

Este projeto implementa uma arquitetura analítica moderna baseada no padrão **Lakehouse / Modern Data Stack**, utilizando:

- Fonte transacional (AdventureWorks)
- Camada analítica Gold (Star Schema)
- Banco PostgreSQL em nuvem (Neon)
- Orquestração e catálogo pela Dadosfera
- Visualização via Metabase

A arquitetura foi desenhada para suportar:
- Análises de vendas
- Performance de produtos
- Rentabilidade
- Acompanhamento temporal

---

## Fluxo de Dados

1. **Fonte de dados**
   - Dados do e-commerce AdventureWorks

2. **Camada Gold (Star Schema)**
   - Fato: `TB_PA37IS_GOLD_FACT_SALES`
   - Dimensões:
     - `TB_PA37IS_GOLD_DIM_PRODUCT`
     - `TB_PA37IS_GOLD_DIM_CUSTOMER`
     - `TB_PA37IS_GOLD_DIM_DATE`

3. **Banco Cloud (Neon PostgreSQL)**
   - Armazena todas as tabelas Gold
   - Fonte oficial para consumo analítico

4. **Dadosfera**
   - Orquestra a ingestão
   - Cataloga os datasets
   - Disponibiliza para BI (Metabase)

5. **Metabase**
   - Criação dos dashboards e métricas

---

## Benefícios da Arquitetura

- Baixa latência para BI
- Modelo dimensional (Star Schema)
- Separação clara de fato e dimensões
- Escalável para novos datasets
